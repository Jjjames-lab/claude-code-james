## 💡 使用建议

### 查看笔记
1. 打开 `vietnam_travel_dashboard.html`
2. 按城市或维度筛选
3. 点击感兴趣的笔记卡片
4. 跳转到小红书查看完整内容（图片、详细评价）

### 规划行程
1. 用HTML仪表板浏览所有推荐
2. 将心仪的项目记录下来
3. 导入CSV到飞书
