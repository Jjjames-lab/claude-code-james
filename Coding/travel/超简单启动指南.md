# 🌍 AI旅行顾问 - 超简单启动指南

## 📝 启动步骤（超简单）

### 方法一：双击启动（推荐）

**第1步：启动后端（服务器）**
1. 找到文件：`启动后端.sh`
2. 双击这个文件
3. 会弹出一个窗口，显示一些文字
4. 看到 "AI旅行顾问 - 后端服务已启动" 就成功了
5. **不要关闭这个窗口！**

**第2步：启动前端（聊天界面）**
1. 再打开一个新的终端窗口
2. 找到文件：`启动前端.sh`
3. 双击这个文件
4. 会自动打开浏览器
5. 看到聊天界面就成功了！

---

### 方法二：命令行启动

如果双击不工作，用这个方法：

**第1步：打开"终端"**
- 按 `Command + Space`，输入 "终端"，回车

**第2步：启动后端**
```bash
cd /Users/tbingy/Desktop/Claude\ Code/Coding/travel
./启动后端.sh
```
**不要关闭这个窗口！**

**第3步：再打开一个新终端**（同样的方法：Command + Space → "终端"）

**第4步：启动前端**
```bash
cd /Users/tbingy/Desktop/Claude\ Code/Coding/travel
./启动前端.sh
```

## 🤖 如何使用

1. 看到AI小星的问候："你好！我是你的AI旅行顾问小星..."
2. 在下方输入框输入你的话，比如："我想去越南玩7天"
3. 点击"发送"按钮
4. AI会回复你，右边的信息栏会自动更新

## ⚠️ 常见问题

### Q: 双击 .sh 文件没反应？
A: 用命令行启动（方法二）

### Q: 提示 "command not found"？
A: 需要先安装 Node.js，下载地址：https://nodejs.org/

### Q: 打开浏览器但看不到界面？
A:
- 检查后端窗口是否还在运行
- 等待10-20秒，前端正在启动
- 如果还不行，刷新浏览器

### Q: AI回复很慢？
A: 第一次启动会慢一些，因为要连接到GLM API，请耐心等待

## 🛑 如何停止

**停止前端**：
- 在前端的终端窗口按 `Control + C`

**停止后端**：
- 在后端的终端窗口按 `Control + C`

**关闭浏览器**：
- 直接关闭浏览器窗口就行

---

## 💡 提示

- 启动时，**后端必须先启动，前端才能工作**
- 两个终端窗口都不要关闭
- 第一次运行会安装依赖，可能需要几分钟
- API Key已经配置好了，不用担心
