## 使用指南

### 如何添加新记录？

在**"最新工作记录"**下方添加新条目，格式：

```markdown
### [YYYY-MM-DD HH:MM] 任务：[简短标题]

**状态**：✅ 已完成 | 🟡 进行中 | 🔴 阻塞中

**任务描述**：
[用户给的原始指令，或者任务的简要说明]

**完成内容**：
- [x] 已完成的工作项1
- [x] 已完成的工作项2
- [ ] 待完成的工作项3

**产出物**：
- 文件：`路径/文件名`
- 说明：[文件内容说明]

**遇到的问题**：
- 问题1：[描述]
- 解决方案：[描述]

**技术决策**：
- 决策1：[描述]
- 理由：[描述]

**对其他角色的建议**：
- 给[角色名]：[建议内容]

**下一步任务**：
- [ ] 等待用户分配下一个任务
```

### 何时归档？

当"最新工作记录"超过3条时，将旧记录移到"工作历史"部分。

### 状态说明

- **✅ 已完成**：任务已完成，产出物已就位
- **🟡 进行中**：正在执行任务
- **🔴 阻塞中**：等待前置依赖，无法继续
