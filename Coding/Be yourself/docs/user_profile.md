## 核心痛点

### 1. 内耗严重

**表现形式**：
- 行动前内耗，怕出错、怕做不好
- 在乎别人评价，渴望认可
- 信息过载、情绪过载会压垮
- 不善于面对冲突，生气时表达不出来

**深层原因**：
- 高敏感 + 高标准 + 低自我认可
- 不是懒，是害怕失败

**产品应对**：
- 先承接情绪，不急于给建议
- 蹲下来平视，不居高临下
- 帮助用户看见自己的成长

### 2. 学习碎片化

**表现形式**：
- 多平台学习（X、小红书、公众号、YouTube）
- 收藏很多但行动少
- 知识之间缺乏连接

**深层原因**：
- 没有系统化的学习框架
- 缺乏时间规划和优先级管理

**产品应对**：
- 自动捕获和分类内容
- 建立知识关联网络
- 提供学习计划和提醒

### 3. 目标模糊

**表现形式**：
- 不清楚自己真正想要什么
- 目标经常变化或模糊
- "顶尖PM"可能不是真正目标

**深层原因**：
- 自我认知还在探索中
- 对"成功"的定义不清晰

**产品应对**：
- 定期帮助用户梳理目标
- 区分"表面目标"和"深层需求"
- 跟踪目标变化
